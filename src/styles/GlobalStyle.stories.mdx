# Global Style

CSS resets and base stylesheets definition.

```jsx
const App = () => (
  <>
    <GlobalStyle />
    <OtherComponents />
  </>
);
```

## Typography

Basic global Typography.

### Headings

All HTML headings, `<h1>` through `<h6>`, are available:

<Preview>
  <Story name="Headings">
    <div>
      <h1>Titulo 1</h1>
      <h2>Titulo 2</h2>
      <h3>Titulo 3</h3>
      <h4>Titulo 4</h4>
      <h5>Titulo 5</h5>
      <h6>Titulo 6</h6>
    </div>
  </Story>
</Preview>
